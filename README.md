# Cargo Management System

## About

Cargo Management System is a console-based Java application developed to simulate the basic operations of a cargo company.

The application allows users to create and manage cargos, customers, and delivery vehicles. It also supports cargo tracking, vehicle assignment, cargo status updates, and customer-based cargo searches.

The project was developed to improve Java programming skills while practicing Object-Oriented Programming principles and Java Collections.

---

## Features

- Create new cargos
- Track cargo by tracking number
- Search cargos by customer ID
- Update cargo status
- Remove cargos
- Add and remove delivery vehicles
- Assign cargos to vehicles
- Prevent duplicate cargo and vehicle records
- Display cargos assigned to a vehicle
- Console-based menu interface
- Input validation for user entries
