# Kargo Yönetim Sistemi

## Proje Hakkında

Kargo Yönetim Sistemi, bir kargo firmasının temel işlemlerini simüle etmek amacıyla geliştirilmiş konsol tabanlı bir Java uygulamasıdır.

Uygulama sayesinde kargo oluşturma, araç yönetimi, kargo takibi, araçlara kargo atama, müşteri bazlı sorgulama ve kargo durumlarını güncelleme işlemleri gerçekleştirilebilmektedir.

Proje, Java programlama becerilerini geliştirmek ve Nesne Yönelimli Programlama (OOP) ile Java Collections yapılarını gerçek bir senaryo üzerinde uygulamak amacıyla hazırlanmıştır.

---

## Özellikler

- Yeni kargo oluşturma
- Takip numarasına göre kargo sorgulama
- Müşteri kimlik numarasına göre kargo sorgulama
- Kargo durumunu güncelleme
- Kargo silme
- Araç ekleme ve silme
- Araçlara kargo atama
- Aynı kargo veya aracın tekrar eklenmesini engelleme
- Araca yüklenen kargoları listeleme
- Menü tabanlı konsol arayüzü
- Kullanıcı girişleri için temel doğrulamalar
